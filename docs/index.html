Multi-Monitor Management
Made Easy

MonitorFusion gives you per-monitor wallpapers, hotkey window movement, and edge snapping — all in a lightweight open-source app. A modern alternative to DisplayFusion.

Download for Windows View on GitHub

Requires Windows 10/11 · .NET 8 Runtime

Everything you need for a multi-monitor setup

Built with C# and WPF, designed to be fast, lightweight, and extensible.

🖼️

Per-Monitor Wallpapers

Set a different wallpaper on each display. Save and switch between wallpaper profiles instantly.

⌨️

Hotkey Window Movement

Move any window to the next or previous monitor with a single keypress. Fully configurable shortcuts.

🧲

Window Snapping

Windows snap to monitor edges and to each other, making it easy to organize your workspace.

📐

Monitor Profiles

Save and restore complete monitor configurations — resolution, position, and orientation.

💾

Window Position Profiles

Save all open window positions and restore them when you reconnect your displays.

🪶

Lightweight by Design

Built on .NET 8 with a minimal footprint. Lives quietly in your system tray until you need it.

Roadmap

MonitorFusion is in active development. Here's what's planned.

Foundation

Project structure & architecture

Core services, Win32 interop layer, WPF shell, system tray icon.

Phase 1

Monitor detection + basic wallpaper

Enumerate displays, detect changes, set per-monitor wallpapers via IDesktopWallpaper COM.

Phase 2

Wallpaper profiles & rotation

Named profiles, image sizing modes, random rotation from folder, online sources.

Phase 3

Window management & snapping

Global hotkeys, move windows between monitors, snap to edges, save/restore layouts.

Phase 4

Monitor profiles

Save/load display configurations, auto-switch on display change.

Phase 5

Multi-monitor taskbar

Custom taskbar on secondary monitors with live thumbnails and pinned shortcuts.

Phase 6

Installer, auto-update & polish

InnoSetup installer, Squirrel auto-updates, settings UI refinements.
